# CMSC207_Mini-Project
This project is a simple website About Me built using **HTML**, **CSS**, and embedded **Java code** within the HTML file.  
It demonstrates how markup, styling, and Java logic can be combined to create interactive web content.

## 🚀 How to Run
1. Download or clone the project files.
2. Open `index.html` in any web browser (e.g., Chrome, Firefox, Edge).
3. Fun-facts.html can interactively be opened inside index.html
4. The embedded Java code will run directly within the HTML file.

## 📂 Project Structure
- `Home.html` → Main webpage containing structure and embedded Java code.
- `Fun-facts.html` → Additional webpage containing other information linked and embedded interactively within ##Home.html.
- `style.css` → Stylesheet controlling layout, colors, and design.
- `images/` → Folder containing image assets used in the project.

## ✨ Features
- Clean and organized HTML structure.
- CSS styling for layout and design.
- Embedded Java code for logic and interactivity.
- Demonstrates integration of markup, style, and programming logic in one file.

## 🛠️ Technologies Used
- HTML5
- CSS3
- Java (embedded in HTML)

## 📚 Learning Objectives
- Practice writing semantic HTML.
- Apply CSS rules for styling and layout.
- Understand how HTML and CSS work together to build a webpage.
- Explore embedding Java code within HTML for interactive functionality.
